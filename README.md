## Nike Store

Complete MERN Full-Stack application built with:

| Front-end     | Back-end     |
|--------------  |--------------|
| Next 13        | Node.js      |
| TypeScript     | Express.js   |
| Material UI 5  | cors         |
| SCSS           | Mongoose     |
| Jotai          | MongoDB      |
| Axios          |              |

- Check here: https://frontend-shoes-store.vercel.app
- Back-end repository: https://github.com/thiagosaraiva-git/nike-store-api
